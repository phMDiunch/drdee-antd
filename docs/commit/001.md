# DRDEE-ANTD Project Setup Guide

## 1. Initialize Next.js Project

```bash
npx create-next-app@latest drdee-antd
```

## 2. Install Ant Design & Icons

```bash
npm install antd @ant-design/icons
```

## 3. Fix React 19 Compatibility Issues

### Install compatibility patch:

```bash
npm install @ant-design/v5-patch-for-react-19
```

### Import compatibility package in `app/layout.tsx`:

```typescript
import "@ant-design/v5-patch-for-react-19";
```

## 4. Setup Ant Design Registry

### Install registry:

```bash
npm install @ant-design/nextjs-registry
```

### Wrap children in `app/layout.tsx`:

```typescript
<AntdRegistry>{children}</AntdRegistry>
```

## 5. Setup Supabase Authentication

### Install Supabase packages:

```bash
npm install @supabase/supabase-js @supabase/ssr
```

### Environment variables:

Create `.env.local` file with Supabase credentials

### Create service files:

- `src/service/supabase/client.ts`
- `src/service/supabase/server.ts`
- `src/service/supabase/middleware.ts`
- `src/middleware.ts`

## 6. Setup Prisma & TypeScript Execution

```bash
npm install prisma tsx --save-dev
npm install @prisma/client
```
